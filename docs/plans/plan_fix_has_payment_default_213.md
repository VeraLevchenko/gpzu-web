# Задача: Исправить флаги has_payment_default в object_types.json согласно Постановлению №213

## Контекст

Пользователь запросил проверку флагов платности (`has_payment_default`) по PDF-файлу Постановления Коллегии Администрации Кемеровской области №213 от 01.07.2015 (с изменениями на 27.05.2025), расположенному по пути `/root/projects/gpzu-web/backend/data/datatest/213.pdf`.

Согласно п.22 Постановления №213, использование земельных участков для следующих объектов является **бесплатным** (free list):
- п.4 (кроме парковок), п.4.1, п.6 (кроме нефтепроводов DN>300), п.12, п.13, п.18, п.19 (кроме кафе), п.21, п.26, п.27, п.29

Все остальные объекты **платные**. Объекты пп.33, 34, 34.1 (ФСИН, ФСБ) оставляем `false` — они используются только государственными структурами, коммерческая плата не применяется.

Изменяется только один файл: `backend/config/object_types.json`.

## Файл для изменения

`backend/config/object_types.json` — изменить поле `"has_payment_default"` в записях с указанными номерами.

---

## Проверка

```bash
cd /root/projects/gpzu-web/backend
python3 -c "
import json
d = json.load(open('config/object_types.json'))
changed = ['8','12','13','15','18','22','24','26','27','28','31','31.1','35','37','39']
for o in d:
    if o['number'] in changed:
        print(o['number'], o['has_payment_default'], o['short_name'][:40])
"
```

Ожидаемый результат:
- пп. 12, 13, 18, 26, 27 → `False`
- пп. 8, 15, 22, 24, 28, 31, 31.1, 35, 37, 39 → `True`
