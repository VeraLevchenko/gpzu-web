# План: Генерация решения о выдаче разрешения РРР (динамический шаблон)

## Контекст

Текущий `rrr_decision_builder.py` использует простой шаблон с плейсхолдерами без условной логики. Реальные решения отличаются в зависимости от вида объекта: меняются правовые основания, появляется/исчезает раздел оплаты, варьируются условия. Необходимо переработать генерацию, чтобы один шаблон с Jinja2-условиями покрывал все варианты.

## Спецификация

### 1. Правовые основания (блок «На основании...»)

**Базовые (все решения):**
- ст.39.36 ЗК РФ
- ПП РФ от 03.12.2014 №1300
- ПП КО от 01.07.2015 №213
- Правила благоустройства №16/198
- Положение о Комитете

**Условные (автоопределяются по виду объекта):**
- `{% if is_linear %}` (пп.1-3, 5-8, 32) → + ст.51 ГрК РФ + ПП РФ от 12.11.2020 №1816
- `{% if is_nto %}` (пп.19, 23-25, 36) → + Постановление администрации г. Новокузнецка от 11.04.2018 №63
- `{% if proezd_agreement %}` (п.9, текст вводится вручную) → + согласование на примыкание

### 2. Оплата (по п.22 Постановления 213)

**Общее правило**: ВСЕ объекты ПЛАТНЫЕ, за исключением:
- Бюджетных проектов, госорганов, муниципальных учреждений
- Пунктов: **4** (кроме парковок/стоянок/автостоянок по наименованию), **4.1**, **6** (кроме нефтепроводов), **12**, **13**, **18**, **19** (кроме сезонных кафе), **21**, **26**, **27**, **29**

Платность для п.4 определяется по `object_name` — платно только если содержит «парковка», «стоянка», «автостоянка».

**Формула расчёта (все платные кроме п.5):**
```
П_годовой = Су × Нст × S
П_итого = П_годовой × (дней_периода / 365)
Где: Су = 899,98 руб./кв.м, Нст = 1,5%, S = площадь (из карточки)
Период = decision_date → end_date (точное кол-во дней)
```

**Для п.5 (ЛЭП до 35кВ) — п.23-1:**
```
П_годовой = 10,06 × S × Ки2026
П_итого = П_годовой × (дней_периода / 365)
Где: Ки2026 = 1,04 (4% инфляция)
```

Константы в `payment_config.json`. При наличии оплаты: п.3 с реквизитами, Приложение №2 «РАСЧЕТ ПЛАТЫ», подпункт «невнесения платы» в прекращении.

### 3. Прекращение действия решения

**Базовые:** 1) истечение срока, 2) ликвидация, 3) предоставление ЗУ, 4) ненадлежащее использование

**Условные:** `{% if has_payment %}` → невнесение платы; `{% if is_nto %}` → отсутствие свидетельства аттракциона

### 4. Условия размещения

**Статичные** (в шаблоне для всех): уведомление о ликвидации, земляные работы, геосъёмка, согласования РСО, права третьих лиц, вырубка, очистка, восстановление, содержание, учёт ЗОУИТ, контроль

**Условные:** `{% if is_tech_connection %}` (пп.1-3, 5-7) → тех.присоединение; `{% if red_lines_inside_area > 0 %}` → проект планировки дорожной сети

### 5. Срок действия

Из заявления (`term_months`). Проверка: max **5 лет** (пп.1-3, 5-7, 11), max **3 года** (остальные). Формат: «36 (тридцать шесть) месяцев»

### 6. Заявитель

**ЮЛ**: наименование, ИНН, ОГРН, адрес → «(далее — Заинтересованное лицо)»
**ФЛ**: ФИО, паспорт, адрес → «(далее — Заинтересованное лицо)»

### 7. Подпись

Статичный блок в шаблоне. Подготовлено — из авторизованного пользователя.
