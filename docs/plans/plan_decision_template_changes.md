| `{{ formula_params.su \| string \| replace('.', ',') }}` | Ставка Су (стандартная) |

| `{{ PAYMENT_YEARLY }}` | Годовой платёж (число) |
| `{{ PAYMENT_YEARLY_WORDS }}` | Годовой платёж (прописью) |
| `{{ PAYMENT_TOTAL }}` | Итоговый платёж (число) |
| `{{ PAYMENT_TOTAL_WORDS }}` | Итоговый платёж (прописью) |
| `{{ PERIOD_DAYS }}` | Количество дней периода |
| `{{ N_TERM_PAYMENT }}` | Номер пункта о прекращении за неоплату |

### Что НЕ ИСПОЛЬЗУЕТСЯ в шаблоне (но доступно из backend):

| Переменная | Значение | Когда нужно |
|-----------|---------|-------------|
| `{{ N_PAYMENT }}` | `3` (если платный) | Если в тексте нужна ссылка на номер пункта оплаты (сейчас, вероятно, жёстко написано "3.") |
| `{{ SU }}` | число | Дублирует `formula_params.su` — можно не менять |
| `{{ NST_PCT }}` | строка, напр. `"1.5"` | Дублирует `formula_params.nst` — можно не менять |
| `{{ BASE_RATE }}` | число | Дублирует `formula_params.base_rate` — можно не менять |
| `{{ KI }}` | число | Дублирует `formula_params.ki` — можно не менять |
| `{{ AREA }}` | строка, напр. `"125,3"` | Площадь отформатированная (с запятой) |

## Что нужно изменить в шаблоне

### 1. Номер пункта оплаты (если жёстко вписано "3.")

Найти в тексте шаблона строку вида:
```
3. Установить плату за размещение объекта...
```
Заменить `3.` на `{{ N_PAYMENT }}.`

Это нужно, чтобы при добавлении новых пунктов выше нумерация не сбивалась.

### 2. Площадь в формуле расчёта

Если в блоке `{% if has_payment %}` площадь выводится как число (напр. вставлено вручную), заменить на:
```
{{ AREA }} м²
```
(уже содержит запятую вместо точки)

### 3. Структура блока расчёта (если нет — добавить целиком)

Если в шаблоне внутри `{% if has_payment %}` нет самого расчёта, добавить:

```
{{ N_PAYMENT }}. Установить плату за размещение объекта.

{% if payment_formula == 'lep' %}
Размер платы рассчитан по формуле: Б × S × Ki
Б (базовая ставка) = {{ formula_params.base_rate | string | replace('.', ',') }} руб./м²
Ki = {{ formula_params.ki | string | replace('.', ',') }}
S (площадь) = {{ AREA }} м²

Годовой размер платы: {{ PAYMENT_YEARLY }} руб. ({{ PAYMENT_YEARLY_WORDS }})
Итого за {{ PERIOD_DAYS }} дней: {{ PAYMENT_TOTAL }} руб. ({{ PAYMENT_TOTAL_WORDS }})
{% else %}
Размер платы рассчитан по формуле: Су × Нст × S
Су = {{ formula_params.su | string | replace('.', ',') }} руб./м²
Нст = {{ formula_params.nst | string | replace('.', ',') }}
S (площадь) = {{ AREA }} м²

Годовой размер платы: {{ PAYMENT_YEARLY }} руб. ({{ PAYMENT_YEARLY_WORDS }})
Итого за {{ PERIOD_DAYS }} дней: {{ PAYMENT_TOTAL }} руб. ({{ PAYMENT_TOTAL_WORDS }})
{% endif %}
```
